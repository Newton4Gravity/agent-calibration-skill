# Release Notes

## v2.1 — Evidence Authority Release

Initial public release of Agent Calibration Skill.

This release includes:

- Universal system prompt
- Evidence Authority Ladder
- Confidence labels: `[VERIFIED]`, `[INFERRED]`, `[ASSUMED]`, `[UNKNOWN]`
- Agent type classification
- Calibration test
- Recovery protocol
- Failure mode reference
- Quick reference card
- Session log template
- Examples for ChatGPT, Claude Code, Cursor, local agents, and research agents
