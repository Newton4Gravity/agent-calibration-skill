# Changelog

## 2.1 — Evidence Authority Release

- Added Evidence Authority Ladder.
- Reframed calibration around claim-specific evidence authority.
- Replaced rigid “raw tool result must always be visible” language with auditable evidence standard.
- Made the default tool budget flexible: produce a partial useful result after about four focused calls, but allow more with justification.
- Added high-impact action safety and authorization rule.
- Expanded failure modes with Scope Drift and Unsafe Action Without Authorization.
- Sanitized project wording for public release.

## 2.0 — Universal Release

- Generalized from domain-specific agent calibration to any agent, any domain, any platform.
- Added three broad agent types.
- Added confidence labeling standard.
- Added universal system prompt.
- Added quick reference card and session log template.
